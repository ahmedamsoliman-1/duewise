# Duewise product plan

## Near-term focus

The next wave of improvements should make Duewise feel less like a record book and more like a real household operations system.

### Phase 1 — recurring workflows
- Add recurring task support with automatic follow-up generation.
- Add recurring subscription settings with interval and end-date controls.
- Make the workflow feel useful from the first week of use.

### Phase 2 — templates and quick starts
- Expand the common workflow templates already present in the resource pages.
- Add more role-based and household-based templates.
- Let users save custom templates for repeated admin patterns.

### Phase 3 — bulk and batch actions
- Support multi-select actions such as complete, delete, and duplicate.
- Add copy/duplicate actions for tasks, subscriptions, and documents.
- Reduce repetitive manual input for similar items.

### Phase 4 — ordering and prioritization
- Add better drag-and-drop ordering for tasks and lists.
- Support custom grouping or pinned priorities.
- Make it easier to organize what matters most.

## Initial implementation order
1. Recurring task follow-ups
2. Recurring subscription settings
3. Duplicate and copy actions
4. Bulk selection and ordering improvements
